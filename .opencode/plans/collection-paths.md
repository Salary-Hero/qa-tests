# API Collection Paths

Fill in these paths once. The `analyse-feature` command reads this file to locate endpoint definitions.

## Postman Collection

```
POSTMAN_COLLECTION_PATH=/Users/admin/Workspace/
```

Replace the value with the absolute path to the folder containing your Postman collection JSON file(s).

## Bruno Collection

```
BRUNO_COLLECTION_PATH=/Users/admin/Workspace/
```

Replace the value with the absolute path to the root Bruno collection folder.

---

**Note:** These paths are personal and gitignored. Do not commit this file.
